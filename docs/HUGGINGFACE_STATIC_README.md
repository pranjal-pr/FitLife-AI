# FitLife AI

This Static Space hosts the FitLife Next.js frontend maintained by
[pranjal-pr](https://github.com/pranjal-pr).

The API is hosted separately because free Hugging Face Static Spaces cannot
run the Python/Flask backend.

The project is based on the original
[FitLife-AI project](https://github.com/shinzoxD/FitLife-AI) by Nishchay
